3D Model Animation Test

Testing the enhanced 3D model with initial animation and interaction

Expected Behavior:

  1. Initial Animation: The model should play an initial animation (either built-in or camera animation) for 3 seconds
  2. Loading Indicator: A spinning loader should appear while the model loads
  3. Auto-rotation: After the initial animation, the model should start spinning autonomously
  4. User Interaction: You can drag to rotate, scroll to zoom, and the auto-rotation will pause during interaction
  5. Resume Auto-rotation: 2 seconds after you stop interacting, auto-rotation should resume
